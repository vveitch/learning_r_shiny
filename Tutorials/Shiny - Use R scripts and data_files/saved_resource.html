                    Edit: Still struggling with this problem. Additional things I've unsuccessfully tried:

                    - Replacing the source() command with all lines of the source files.
                    - Wrapping the creation of sTableTypes in a function that is called in the script file.
                    - Wrapping the creation of sTableTypes in a function that is called as a reactive object.

                    Since I used sTableTypes in both server.R and ui.R, I had to source it in both. Sourcing manually created it in a global environment, so that explains why it worked. This can also be done by creating a global.R file, which will create objects accessible to both ui and server.

                    Hopefully this helps others struggling with this.
