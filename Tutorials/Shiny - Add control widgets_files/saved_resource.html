                          Andrej Oskin sidpat 5 months ago

                          Concerning your second question. It seems, that the problem that shiny developers do not use 'placeholder' attribute. Hope they will fix it in future releases. Right now you can use following hack:

                          In ui.r, after library(shiny) you should add:

                          textInput <- function (inputId, label, value = "", placeholder = NULL)
                          {
                          if (is.null(placeholder)) {
                          tagList(label %AND% tags$label(label, `for` = inputId), tags$input(id = inputId,
                          type = "text", value = value))
                          } else {
                          tagList(label %AND% tags$label(label, `for` = inputId), tags$input(id = inputId,
                          type = "text", value = value, placeholder = placeholder))
                          }
                          }

                          environment(textInput) <- environment(sliderInput)

                          Then textInput usage should be modified as follows:

                          textInput("text", label = h3("Text input"),
                          value = "",
                          placeholder = "Enter text...")

                                                                Garrett Mod kushdesh 5 months ago

                                                                Widgets just collect data and pass on data, which means that you can use a widget to select anything in R. Most R functions that use color speak about the color as a character string. For example, to make a red histogram you might write hist(mtcars$mpg, col = "red"). To control this color, you only need a widget that returns character strings. For example, if you build a select box widget named foo that returns one of several color names (as a character string: "red", "green", "blue"), then you can use it to control your histogram color with renderPlot({hist(mtcars$mpg, col = input$foo)}).
